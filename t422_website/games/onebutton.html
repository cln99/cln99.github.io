This is probably the best game I've made this term, since I was actually able to get it done and released. I made this for the One Button Jam in which the only restriction is that your game can only use one button.

In the game you have to escape a cave while evading a ghost, you can optionally collect coins as well.

I had the idea of a platformer where you control by hitting a button which takes you forward and up and touching a wall flips you in the other direction. I think there was a mobile game that had this concept but I can't remember what it was.

After making the player controller I originally wanted it to be a level based game but decided against it instead going for one big cave to explore and collect coins. I wanted it to be about facing platform challenges and escaping the cave but I found it hard to make interesting challenges and sections with the players movement so I decided to make it about escaping the big cave with a ghost chasing you instead.

I'm pretty happy with the ghosts movement, it uses a modified version of a planes movement from a previous game I was working on and it works very well, the ghost is very unpredictable and always seems to catch you at the wrong time.

I released this game 10 hours before the deadline and I couldn't think of anything else to add or polish. Releasing it felt really nice as I haven't finished a game in a while and I'm really happy to have atleast one game done.
