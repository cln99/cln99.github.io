Mini Jam 111: Colors

JAM PAGE

RELEASE PAGE

The Mini Jam is a biweekly jam that lasts 3 days, there is an optional theme and a required limitation on the games submitted, this time the theme was Colors and the limitation was "You are your own enemy."

This was my most successful jam as I actually managed to get something done and submitted. I'm very happy with how it ended up and makes me want to do more short jams in the future.

My idea starting out would be some kind of mail sorting with colors going from one place to another, I then used this base idea to make a top down 2d game where you deliver mail in a randomly generated town while avoiding the angry townspeople.

My only regret is that I wasn't really able to include the limitation in it since it just didn't really work with what I was making.
