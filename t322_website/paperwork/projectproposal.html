Project Proposal

Term 3 Project Proposal 2022

Introduction

My project this term isn't one specific project but several mixed together. I want to have one "big" project that is similar to what I've done before but alongside several smaller projects. These smaller projects will mostly be smaller game projects and maybe some miscellaneous programming projects if I happen to do some.

I plan to do a couple game jams throughout the term since I feel it would be a very good way to get feedback and have a deadline to work towards outside of school. I also want to improve my planning and work ethic by better journaling and planning my days out, using planning software.

Description

This term I will be making a couple games in the "Mini Jam 111" (July 22nd to July 25th), "LOW REZ JAM" (August 1st to August 14th) and the “Metroidvania Month” (August 15th to September 15th).

Development

This term I want to improve my planning since it has been a weak point of my projects that I haven't quite tackled properly, using software like Focalboard which I hope will help my planning and project management.

Along with this I hope to release and get feedback on the games I make this term, as I haven’t properly released anything to the public yet.

Learning Goals

Personal Qualities through further polishing my game development and programming skills, although especially this term I hope to improve my project management.

Communication through getting feedback and critique on my game, especially through my game jam games since those will be judged and ranked by a lot of people.

Quantitative Reasoning through programming, problem solving and math related mechanics. For example, vector math is something I learn through game development.

Empirical Reasoning through game design and experimenting with game mechanics.

Social Reasoning through again getting feedback and critique as well as doing various game reviews.

Knowing how to learn through creating and refining my knowledge of game development.

Five A's

Authenticity

I chose this kind of project since last term I chose a large project and it didn't pan out that well, due to poor planning, not really having an idea on what to do and little to no playtesting. I feel that having smaller projects and participating in game jams will help me improve more and gain more expertise in game development.

Active Learning

Since I will participate in game jams I will be working on my games a lot outside of school, especially the Mini Jam since that takes place on the weekend only. I will be using the same engine and tools that I have been using since the start of the year.

Adult Relationships

I don't have any mentors, but I will be getting feedback and responses from players and game developers.

Academic Rigor

This term will probably be a big one for me since it pushes me into more sides of game development I don't have much experience with, most notably finishing and releasing games.

Assessment

Through game jams my games will be critiqued and rated which will give me a lot of insight and help me improve my future games.

Portfolio

I plan to make a website for my portfolio so it's easier and nicer to traverse than a google drive, it also allows me to make the overall documents look better. It also allows me to embed the games into the site so you can play them without downloading anything. Evaluation I feel fairly confident about this term although it could go very wrong, I just hope I don't get too burnt out or exhausted from the game jams.
