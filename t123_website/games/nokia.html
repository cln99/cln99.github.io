GAME PAGE

The Nokia Jam is a yearly jam dedicated to making games in the style of an old Nokia cellphone. These restrictions are a low resolution (84x48) and a two color palette.

The game I ended up making for this jam was dunvon, a very simple isometric dungeon crawler. This was my first time using isometric tiles and graphics which turns out are pretty cool.

The gameplay itself is really simple, you go around a randomly generated dungeon to find a hole to get to the next floor, there are enemies that you can attack and coins to collect.

Overall, a really simple game but I'm happy I got it done, also for whatever reason I got a lot of feedback and responses (mostly positive!) which you can see on the game page.
