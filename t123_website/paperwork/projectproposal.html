Term 4 Project Proposal 2022

Introduction

This term I will be doing a similar project to last term, where I participate in game jams and release games over the course of the term. I hope to improve on my shortcomings last term and hone my game making skills further.

I plan on only entering 1-3 day game jams since I feel I can make better use of a shorter deadline since it limits my scope which makes finishing them easier. I also only want to work on my games on the weekend to allow me to focus more on development, as I found working during school was difficult.

Description

This term I will be making 1-3 games (depending on the game jams happening during weeks 4-8) for various game jams and releasing them to get feedback and criticism.

Five A’s

Authenticity

I chose this project because making games lets me gain knowledge and skill in a bunch of different areas, most notably programming and game design, but also in project management, art, sound design and other areas.

Active Learning

I plan to develop mostly during the weekend as I can focus more and don’t have to worry about other work that I do during the week. I’ll be using the same tools as I have in previous projects being Godot Engine for making the game and Aseprite or Krita for art, and a bunch of other tools for niche uses.

Adult Relationships

I don't have any mentors, but I will be getting feedback and responses from players and game developers.

Academic Rigor

Like last term, this project further pushes me into finishing and releasing games, which for me is the hardest part about game development.

Assessment

I will be getting feedback on all of my game jam games by players and other game developers. I’ll hopefully be able to show around 2-3 games, their development process, any feedback I get and my reflection on them.

Portfolio

I plan to use the website I made last term to host my portfolio as it works as a really useful hub of information on my project.

Evaluation

Generally for this term I hope to submit a game for each game jam I enter, as I wasn’t able to submit for most of them last time. I also hope I can manage a research paper of some kind, even if it isn’t a traditional kind of research paper.
